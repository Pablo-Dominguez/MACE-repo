Introduccin

Por su riqueza en protenas, carbohidratos, lpidos y minerales las semillas de calabaza son un producto que llega a ser consumido a lo largo de pases en todo el mundo. Los factores que determinan el tipo concreto de cada semilla son generalmente relativos a la apariencia externa de la misma. Distinguir el tipo de cada semilla suele requerir experiencia y tiempo, cosa que en nuestro mundo moderno se traduce como un coste econmico.

Con el objetivo de aportar una nueva manera de realizar esta tarea, nos proponemos plantear un algoritmo de clasificacin binaria tipo SVM que, a partir de la base de datos Pumpkin_Seeds_Dataset.xlsx, distinga entre dos categoras diferentes de semillas de calabaza.

As pues, comenzaremos este estudio con un anlisis exploratorio de los datos, para seguir con una construccin de un clasificador a partir de este algoritmo y acabar con una evaluacin de los diferentes resultados obtenidos.
